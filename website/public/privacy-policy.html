Privacy Policy

Last updated: May 8, 2026

AnnHub ("we", "our", or "the extension") is a browser extension that provides text highlighting, annotation, and vocabulary labeling on web pages. We are committed to protecting your privacy.

1. Data We Collect

AnnHub collects and stores the following types of data locally on your device:

  • Highlighted text — text you select and highlight, along with the page URL, page title, and CSS selector for highlight restoration
  • Clip records — text you collect using Mode A or Mode B, with surrounding context (up to 20 characters before/after)
  • User notes — any notes you attach to highlights or clips
  • Vocabulary data — word lists, proficiency levels, and gloss/definition cache for the vocabulary labeling feature
  • Extension settings — your configuration preferences
All data listed above is stored locally in your browser using IndexedDB and chrome.storage.local. We do not operate any servers that receive or store your data.

2. Data Sent to Third Parties

AnnHub may send data to third-party services only when you explicitly configure and enable the following optional features:

  • Eudic (欧路词典) API — If you configure an Eudic API token, AnnHub syncs your vocabulary word lists with the Eudic service. Only word lists and proficiency data are transmitted. No page content or browsing history is sent.
  • LLM API (e.g., OpenAI, GLM, DeepSeek) — If you configure an LLM API key and endpoint, AnnHub may send individual English words along with their surrounding sentence context (up to ~300 characters) to the configured LLM service for contextual word definitions. No full page content, URLs, or browsing history is sent to the LLM.
  • Logseq — If you configure Logseq sync, highlights and clips may be synced to your local Logseq instance.

You can disable any of these integrations at any time from the extension's Settings page.

3. Chrome Web Store Limited Use

AnnHub's use of information received from Chrome extension APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. We use extension permissions only to provide user-facing highlighting, annotation, vocabulary labeling, and optional sync features.

4. Data We Do NOT Collect

  • We do not collect your full browsing history or record every page you visit
  • We do not collect personal information (name, email, etc.)
  • We do not use analytics, tracking, or telemetry
  • We do not serve ads
  • We do not sell or share any data

When you actively highlight or collect text, AnnHub stores the related page URL, page title, selected text, and limited surrounding context so it can restore highlights and show the source of your saved content.

5. Permissions Explained

  • storage — store highlights, clips, vocabulary, and settings locally
  • activeTab / tabs — access the current active tab to open or locate highlights selected by you
  • commands — register keyboard shortcuts (Alt+H, Cmd+Shift+H)
  • sidePanel — display the highlight list in Chrome's side panel
  • alarms — schedule periodic vocabulary sync with Eudic (if configured)
  • host_permissions (<all_urls>) — required to inject the content script on any webpage so you can highlight and annotate text on any site you visit

6. Data Security

All API communications (Eudic, LLM) use HTTPS encryption. API keys and tokens are stored in chrome.storage.local and are never exposed to web pages — only the extension's background service worker has access to them.

7. Data Retention and Deletion

All data is stored locally. You can delete all stored data at any time by:

  • Using the extension's built-in clear/delete features
  • Uninstalling the extension (all local data is removed automatically)

8. Children's Privacy

AnnHub is not directed at children under 13. We do not knowingly collect information from children.

9. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated date.

10. Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
